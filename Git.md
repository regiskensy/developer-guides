# Cartilha Git

Esse documento descreve os principais comandos do Git e sua integração com Github

## Configurações globais

* Nome de usuário:

```bash
$ git config --global user.name "username"
```

* E-mail:

```bash
$ git config --global user.email "email@domain.com"
```
